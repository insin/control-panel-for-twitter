# Birdfeed

![](icons/icon128.png)

**A browser extension for desktop and mobile that gives you back control of your Twitter/X experience — less noise, more signal.**

> Birdfeed is an actively maintained continuation of [Control Panel for Twitter](https://github.com/insin/control-panel-for-twitter) by [@insin](https://github.com/insin), which seems to only get sporadic updates. For now, all original features are preserved.

Check the [releases page](https://github.com/fabiogaliano/control-panel-for-twitter/releases) for the latest version and browser availability.

## Status

One-person project. Issues get looked at. PRs get reviewed. Twitter/X breaks things constantly — fixes ship when they're needed.

If something's broken or you have an improvement, contributions are genuinely welcome.

## Contributing

This project runs on community contributions.

- **Bug fixes**: open a [Pull Request](https://github.com/fabiogaliano/control-panel-for-twitter/pulls) directly
- **New features**: [open an Issue](https://github.com/fabiogaliano/control-panel-for-twitter/issues/new) first so we can discuss before building

## Support

To report a bug, [open an Issue](https://github.com/fabiogaliano/control-panel-for-twitter/issues/new).

Please include:

- The version of Birdfeed you're using
- Your browser and operating system
- Relevant URLs and screenshots if applicable
